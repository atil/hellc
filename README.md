# FPS game prototype

[Gameplay video](https://youtu.be/goYyxI0czU4)

- Made with C++/OpenGL
- Platforms: Windows
- External code used: GLEW/GLFW, stb-image
- Level design tool: Unity with RealtimeCSG plugin
- Features:
    - First person controller with Q3-like physics
    - Point and directional shadows
    - Loading OBJ files
